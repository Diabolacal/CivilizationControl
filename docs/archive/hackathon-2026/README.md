# Hackathon 2026 Archive

This folder preserves historical hackathon and demo-production material from the March 2026 CivilizationControl submission.

Archived documents, scripts, and recording notes here are historical reference only. They are not the current product or operational source of truth unless an active document explicitly links back to them.

Current agents should start with:

- `docs/llm-reference-guide.md`
- `README.md`
- `docs/decision-log.md`
- current docs under `docs/operations/`, `docs/architecture/`, `docs/strategy/`, and `docs/ux/`

Why this archive exists:

- reduce future LLM/context noise
- keep old demo assumptions from steering post-hackathon work
- preserve hackathon submission, recording, and evidence history without deleting it

Archive layout:

- `core/` — historical demo blueprints and proof/evidence matrices
- `demo/` — narration, capture, and demo-production runbooks
- `operations/` — one-off hackathon validation, submission, and evidence appendices
- `recordings/` — historical recording workspace notes and local media outputs
- `scripts/` — historical capture/rehearsal/assembly scripts

Historical scripts in this archive should not be treated as current validation commands. Some assume old recording paths, older Stillness or Utopia-era state, or local media assets that were intentionally never committed.
