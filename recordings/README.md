# Recordings

Pipeline workspace for demo video production. Not committed to git (except this README).

## Subdirectories

- `voiceovers/` — TTS narration MP3s + silence WAV assets (see `docs/demo/tts-generation-guide.md` for naming and inventory)
- `segments/` — Split video segments and voiced versions
- `screenshots/` — Debug screenshots and smoke test captures
- `inserts/` — Pre-captured external insert clips (hostile denial, world footage)

## TTS Audio Files

All generated TTS files and silence assets go in `voiceovers/`. See the [TTS Generation Guide](../docs/demo/tts-generation-guide.md) for:
- Exact filenames (14 narration + 7 silence = 21 files)
- Generation order
- QC checklist
- Editor handoff notes
